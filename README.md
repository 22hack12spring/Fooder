## 技術構成
- Go: 1.18
  - Echo
  - Sqlx
- MariaDB: 10.6
- Docker

## For developer

### Setting
create `.env`file in `root`directory.
```text
API_KEY=<your_api_key>
```
### start
```sh
docker compose up
```
